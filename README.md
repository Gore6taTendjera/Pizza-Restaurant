# Mario & Luigi Pizza Restaurant

Web application with Arduino Uno R3 (smart oven) to monitor temperatures and humidity for managing pizza orders in a restaurant, developed in a group of 3.

# Tech stack
- HTML / CSS / JavaScript
- Flask
- Arduino Uno R3
